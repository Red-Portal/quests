=  을 사용하면 변수를 정의하는 동시에 값을 저장할 수 있다고 했는데,
그게 해보니까 생각보다 어려워서...

이렇게 하시면 됩니다.
모든 char 형 변수는 int 형의 ASCII 코드를 갖고 있다.
따라서 어떤 문자가 입력되든간에, ASCII 값인 int 로 바뀔 수 있음

따라서 변수 정의는 이렇게 하면 됩니다.
대빵 큰 int 형 배열을 선언하면 이게 가상의 메모리 역할을 하게 됩니다
ex) int memory[1000]
변수를 입력 받으면 아래와 같이 하면 됩니다

char input; // 이게 받은 변수명이라고 칩시다
int var = (int)input // 받은 변수를 int로 캐스팅합니다.
( (int) 이렇게 명시적으로 캐스팅하지 않고 그냥 대입해도 암묵적으로 될껍니다)

memory[var]; 하면 그 변수에 해당하는 자리에 접근을 할 수 있습니다.

그러니까 값을 저장하려면 memory[var] = data;
이런 식으로 하면 됩니다.
